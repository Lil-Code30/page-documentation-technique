HTML Introduction

HTML est le langage de balisage standard pour créer des pages Web.

Qu'est-ce que le HTML?

  • HTML signifie Hyper Text Markup Language
  • HTML est le langage de balisage standard pour créer des pages Web
  • HTML décrit la structure d'une page Web
  • HTML est constitué d'une série d'éléments
  • Les éléments HTML indiquent au navigateur comment afficher le contenu
  • Les éléments HTML étiquettent des éléments de contenu tels que "c'est un titre", "ceci est un paragraphe", "c'est un lien", etc.
Un simple document HTML 
                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Page Title</title>
                    </head>
                    <body>
                        <h1>My First Heading</h1>
                        <p>My first paragraph.</p>
                    </body>
                    </html>

Example Explained

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph
Qu'est-ce qu'un élément HTML?

Un élément HTML est défini par une balise start, un contenu et une balise end:

<tagname>Le contenu va ici... </tagname>

Le HTML élément est tout, de la balise de début à la balise de fin:

<h1>Mon Première Rubrique</h1>

<p>Mon premier paragraphe.</p>

Start tag Element ContentEnd tag
<h1> Mon Première Rubrique </h1>
<p> Mon premier paragraphe. </p>
<br> none none

HTML Histoire

Depuis les premiers jours du World Wide Web, il y a eu de nombreuses versions de HTML:

  • 1989 : Tim Berners-Lee invented www
  • 1991 : Tim Berners-Lee invented HTML
  • 1993 : Dave Raggett drafted HTML+
  • 1995 : HTML Working Group defined HTML 2.0
  • 1997 : W3C Recommendation: HTML 3.2
  • 1999 : W3C Recommendation: HTML 4.01
  • 2000 : W3C Recommendation: XHTML 1.0
  • 2008 : WHATWG HTML5 First Public Draft
  • 2012 : WHATWG HTML5 Living Standard
  • 2014 : W3C Recommendation: HTML5
  • 2016 : W3C Candidate Recommendation: HTML 5.1
  • 2017 : W3C Recommendation: HTML5.1 2nd Edition
  • 2017 : W3C Recommendation: HTML5.2
HTML Exemples de Base

Dans ce chapitre, nous allons montrer quelques exemples HTML de base.

Ne vous inquiétez pas si nous utilisons des balises que vous n'avez pas encore apprises.

HTML Documents

Tous les documents HTML doivent commencer par une déclaration de type de document: <!DOCTYPE html>.

Le document HTML lui-même commence par <html> et se termine par </html>.

La partie visible du document HTML est entre <body> et /body>.

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Page Title</title>
                    </head>
                    <body>
                        <h1>My First Heading</h1>
                        <p>My first paragraph.</p>
                    </body>
                    </html>
Le !DOCTYPE Déclaration

Le <!DOCTYPE> declaration représente le type de document et aide les navigateurs à afficher correctement les pages Web.

Il ne doit apparaître qu'une seule fois, en haut de la page (avant les balises HTML).

Le <!DOCTYPE> la déclaration n'est pas sensible à la casse.

Le <!DOCTYPE> déclaration pour HTML5 est:

                
                    <!DOCTYPE html>
HTML Rubriques

Les en-têtes HTML sont définis avec <h1> à <h6> tags.

<h1> définit le titre le plus important. <h6> définit le moins important titre:

                
                    <h1>Titre 1</h1>
                    <h2>Titre 2</h2>
                    <h3>Titre 3</h3>
HTML Paragraphes

Les paragraphes HTML sont définis avec <p> tag:

                
                    <p>Ce texte est un paragraphe.</p>
                    <p>Ce texte est un autre paragraphe.</p>
HTML Liens

Les liens HTML sont définis avec <a> tag:

                
                    <a href="https://www.example.com">Exemple de lien</

La destination du lien est spécifiée dans le href attribut.

Les attributs sont utilisés pour fournir des informations supplémentaires sur les éléments HTML.

Vous en apprendrez plus sur les attributs dans un chapitre ultérieur.

HTML Images

Les images HTML sont définies avec <img> étiquette.

Le fichier source (src), texte alternatif (alt), width, et height sont fournis comme attributs:

                
                    <img src="image.jpg" alt="Image de test" width="100" height="200">
